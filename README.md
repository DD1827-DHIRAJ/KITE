
# Zerodha Auto Trader (SMC Bot)

## Setup Instructions

1. Upload all files to GitHub
2. Create new Render service (type: background worker)
3. Add Environment Variables:
    - KITE_API_KEY
    - KITE_API_SECRET
    - KITE_REQUEST_TOKEN
4. Deploy and you're live!
